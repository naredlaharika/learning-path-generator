# learning-path-generator
A Python based tool that generates step-by-step learning roadmaps based on the user input.

## Description
This project is a Python-based application that generates step-by-step learning roadmaps based on user input.

## Features
- Takes user learning goal as input
- Generates structured learning paths
- Beginner-friendly and easy to use

## Technologies Used
- Python

## How to Run
1. Open the code in Python or Google Colab
2. Run the program
3. Enter what you want to learn (e.g., Python, Web Development)
4. Get your learning roadmap

## Example
Input: Python  
Output: Step-by-step learning path for Python
